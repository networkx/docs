to_pydot¶

to_pydot(N, strict=True)[source]¶

Return a pydot graph from a NetworkX graph N.

Parameters :

N : NetworkX graph

A graph created with NetworkX

Examples

>>> K5=nx.complete_graph(5)
>>> P=nx.to_pydot(K5)
